# UI_FLOW.md

## Overlay Mode (Primary)

1. User presses hotkey
2. Screen dims
3. User selects region
4. OCR runs
5. Result panel appears

Panel:
- JP text
- Copy button
- Save as card
- Retry

## Capture Detail

- Click token → create card
