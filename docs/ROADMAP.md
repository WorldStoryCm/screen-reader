# Japanese Game OCR Overlay — Roadmap

## MVP1
Goal: make capture -> OCR -> copy feel reliable and fast.

### Features
- tray/background app
- global hotkey
- region selection overlay
- local OCR for Japanese
- result panel with copy button
- save capture history
- rerun OCR

### Exit criteria
- works during real gameplay session
- user can capture menu text, item text, and screen labels faster than taking phone photos

---

## MVP2
Goal: turn OCR results into reusable study material.

### Features
- tags on captures
- notes on captures
- manual word selection from OCR text
- internal word/card storage
- link cards to captures
- search/filter captures and cards

### Exit criteria
- user can build a small study deck manually from actual game text
